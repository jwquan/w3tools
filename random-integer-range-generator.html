As the randfunction returns a real number, this can be rounded to produce a random integer. For example,

However, these integers would not be evenly distributed in the range. A better method is to use the function randi, which in its simplest form randi(imax)returns a random integer in the range from 1 to imax, inclusive. For example, randi(4)returns a random integer in the range from 1 to 4. A range can also be passed, for example, randi([imin, imax])returns a random integer in the inclusive range from imin to imax:
